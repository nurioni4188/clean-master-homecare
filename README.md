# Clean Master 스마트 홈케어 예약형 웹 패키지 v1.0

서울·경기 프리미엄 홈케어 예약형 웹페이지 패키지입니다.  
고객 문의를 정리하고 전화·문자 예약 상담으로 연결하는 **정적 웹사이트 + 운영자 상담 메모 + 문서/소개자료 패키지**입니다.

## 1. 포함 파일

```text
index.html                         단독 실행 배포 버전
admin.html                         운영자용 상담 메모 화면
privacy.html                       개인정보처리방침 템플릿
terms.html                         이용약관 템플릿

split_version/
  index.html
  style.css
  script.js
  privacy.html
  terms.html

docs/
  Clean_Master_고객상담_스크립트.docx
  Clean_Master_청소리포트_샘플.pdf

presentation/
  Clean_Master_서비스소개서.pptx
  Clean_Master_서비스소개서.pdf
```

## 2. 서비스 개요

Clean Master는 전화번호가 있는 단순 홍보 페이지가 아니라, 고객 문의를 정리하고 예약 상담으로 연결하는 스마트 홈케어 예약형 웹페이지입니다.

주요 기능은 다음과 같습니다.

- 고객 문의 정리 도우미
- MOCK 상담 결과 생성
- LOCAL MATCH 기반 문의 유형 분류
- 예상 서비스 범위 안내
- 예약 전 확인사항 체크리스트
- 추천 문의 문자 자동 생성
- 결과 복사 기능
- 전화·문자 상담 연결
- 운영자용 상담 메모 화면
- 개인정보처리방침 / 이용약관 템플릿
- 상담 스크립트 / 청소 리포트 샘플 / 서비스 소개서

## 3. 중요한 운영 원칙

본 패키지는 실제 자동 견적, 자동 예약 확정, 자동 결제 기능을 포함하지 않습니다.

웹사이트는 고객의 문의 내용을 정리하고, 상담 전 필요한 정보를 구조화하여 전화·문자 상담으로 연결하는 역할을 합니다.

최종 견적, 일정 확정, 서비스 제공 여부는 운영자가 고객과 직접 상담한 후 결정해야 합니다.

아래 항목은 실제 오픈 전 반드시 수정해야 합니다.

```text

사업자등록번호
대표자명
사업장 주소
```

```text

```

실제 운영 번호로 교체해야 합니다.

## 6. Vercel 배포 설정

Vercel에서 GitHub 저장소를 Import할 때 권장 설정은 다음과 같습니다.

```text
Framework Preset: Other
Build Command: 비워두기
Output Directory: .
Install Command: 비워두기
```

루트에 `index.html`이 있으므로 별도 빌드 없이 정적 웹사이트로 배포할 수 있습니다.

## 8. 문구 정합성 기준

웹사이트와 문서·소개자료는 아래 표현을 기준으로 맞춥니다.

```text
스마트 상담 → 문의 정리 도우미
70,000원부터 → 70,000원~150,000원
100,000원부터 → 100,000원~200,000원
자동 견적/자동 예약 확정 표현 사용 금지
상담 전 정보 정리 보조 기능으로 설명
```

## 9. 주의사항

- 본 패키지는 정적 웹사이트입니다.
- 고객 정보가 서버에 저장되지 않습니다.
- 실제 예약 확정은 전화·문자 상담 후 운영자가 진행해야 합니다.
- 개인정보처리방침과 이용약관은 참고용 템플릿이며, 실제 운영 전 법률 검토가 필요할 수 있습니다.

## 10. 추천 운영 방식

초기에는 다음 방식으로 운영하는 것을 권장합니다.

```text
고객이 웹페이지 방문
→ 문의 정리 도우미 작성
→ 결과 복사 또는 문자 상담 연결
→ 운영자가 전화·문자로 상세 상담
→ 방문 일정 및 견적 확정
```

이 방식은 서버, DB, 결제 기능 없이도 빠르게 운영을 시작할 수 있습니다.
